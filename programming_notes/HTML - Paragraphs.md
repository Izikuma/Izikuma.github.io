the tag `<hr>` adds a line to the page, creating sections on the document
```html
<h1>heading 1</h1>
<p>random text</p>
<hr>
<h2>heading 2</h2>
<p>more random text</p>
<hr>
```
will result in
<h1>heading 1</h1>
<p>random text</p>
<hr>
<h2>heading 2</h2>
<p>more random text</p>
<hr>


the tag `<br>` adds a line break to a paragraph
```html
<p>This is<br>a paragraph<br>with line breaks.</p>
```
will result in
<p>This is<br>a paragraph<br>with line breaks.</p>
